Memodifikasi Kendaraan Konvensional Dengan Sistem Kendali Jarak Jauh

Sistem kendaraan berbasis IoT untuk mengontrol fungsi kendaraan secara realtime melalui aplikasi mobile.

Deskripsi Singkat

Project ini merupakan sistem modifikasi kendaraan konvensional menjadi kendaraan dengan fitur kendali jarak jauh berbasis Internet of Things (IoT). Sistem dikembangkan menggunakan aplikasi mobile berbasis Kodular dan Firebase Realtime Database untuk komunikasi realtime antara aplikasi dan mikrokontroler kendaraan.

Status Project

Completed Project

Project berhasil diselesaikan dan seluruh fitur sistem berfungsi dengan baik pada kendaraan sedan yang digunakan untuk pengujian dan implementasi sistem.

Teknologi

ESP32 IoT Kodular Firebase PHP MySQL HTML CSS JavaScript

Komponen Hardware

  • ESP32
  • Relay Module
  • Smartphone Android
  • Sistem Kelistrikan Kendaraan
  • Power Supply / Step Down

Fitur Sistem

  • Menyalakan dan mematikan mesin kendaraan
  • Mengontrol klakson kendaraan
  • Mengunci dan membuka pintu kendaraan
  • Mengontrol lampu kendaraan
  • Sistem kontrol realtime melalui smartphone

Cara Kerja Sistem

  • Pengguna membuka aplikasi mobile
  • Pengguna menekan tombol kontrol kendaraan
  • Data dikirim ke Firebase Realtime Database
  • ESP32 membaca data secara realtime
  • Relay mengontrol sistem kendaraan
