Sistem Peminjaman Buku

Website perpustakaan berbasis web untuk pengelolaan data buku, data anggota, dan transaksi peminjaman buku menggunakan PHP dan MySQL.

Deskripsi Singkat

Sistem Peminjaman Buku merupakan project website perpustakaan berbasis web yang digunakan untuk membantu proses pengelolaan data buku, anggota perpustakaan, serta transaksi peminjaman dan pengembalian buku. Sistem dibuat menggunakan PHP dan MySQL dengan tampilan dashboard admin untuk mempermudah monitoring data perpustakaan.

Teknologi

PHP MySQL HTML CSS

Fitur Sistem

  • Manajemen data buku perpustakaan
  • Manajemen data anggota perpustakaan
  • Sistem transaksi peminjaman buku
  • Sistem pengembalian buku
  • Dashboard monitoring data perpustakaan
  • Laporan data buku dan transaksi

Tampilan Dashboard

Dashboard admin digunakan untuk monitoring jumlah buku, data anggota, transaksi peminjaman, dan pengembalian buku.

ERD Database

ERD digunakan untuk menggambarkan struktur database dan relasi antar tabel pada sistem perpustakaan.

Jobdesk Saya

  • Membuat tampilan dashboard website perpustakaan
  • Merancang struktur database MySQL
  • Membuat sistem pengelolaan data buku dan anggota
  • Membuat fitur transaksi peminjaman dan pengembalian buku
  • Menghubungkan website dengan database MySQL

Hasil Project

Website perpustakaan berhasil dibuat dengan fitur pengelolaan data buku, data anggota, transaksi peminjaman, dan pengembalian buku. Sistem menggunakan database MySQL untuk menyimpan data perpustakaan dan menampilkan informasi melalui dashboard admin berbasis web.
